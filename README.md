# DigiTools Platform 

##  Description
DigiTools Platform is a React-based web application where multiple digital tools are organized and displayed in a clean and user-friendly interface. It helps users explore different tools in one place.

## ⚙️ Technologies Used
- React
- JavaScript (ES6)
- Tailwind CSS
- DaisyUI

##  Features
- Dynamic tools listing
- Responsive UI using Tailwind & DaisyUI
- Lifting state up for better state management
- Conditional Rendering
- Create Fake Api Data using AI (JSON data)
- React Toastify (NPM Package)
- React icons (NPM Package)
